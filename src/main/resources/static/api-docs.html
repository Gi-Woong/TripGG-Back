👥 사용자 관리 API

/users

🔐 인증 API
GET
/auth/kakao/login
카카오 로그인 시작
📝 응답
HTTP 302 리다이렉트로 카카오 로그인 페이지로 이동
POST
/auth/kakao/login
카카오 로그인 처리
authorizationCode
String
카카오에서 받은 인가 코드
응답 예시:
{ "success": true, "message": "카카오 로그인 성공", "data": { "accessToken": "jwt_token_1_1703123456789", "user": { "id": 1, "kakaoId": "test_kakao_id_001", "nickname": "테스트 사용자", "profileImageUrl": "https://example.com/profile.jpg", "createdAt": "2024-01-01T00:00:00", "updatedAt": "2024-01-01T00:00:00" }, "isNewUser": false }, "timestamp": 1703123456789 }
⚠️ 주의사항
• 모든 API는 ApiResponse<T> 형태로 응답
• 현재 모든 API는 인증 없이 접근 가능 (테스트용)
• CORS: 모든 Origin에서 접근 가능
• 위치 기반 서비스: 채팅방과 장소 검색은 사용자 위치 정보 필요
• 카카오 API: 카카오맵 API 키가 설정되어 있어야 정상 작동
