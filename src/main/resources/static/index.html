🔍 카테고리 검색 (가장 확실한 방법)
• 카테고리만으로도 정확한 결과 반환

📍 위치 기반 카테고리 검색

🔐 인증 & 회원 관리

GET /auth/kakao/login (카카오 로그인)
🔑 JWT 토큰 획득 방법
1. 카카오 로그인 버튼 클릭
2. 카카오 로그인 완료 후 JWT 토큰 복사
3. 위의 "JWT 토큰 테스트" 섹션에 토큰 입력
4. "토큰 설정" 버튼 클릭
📋 POST API 엔드포인트
• POST /auth/kakao/login - 카카오 인가코드로 로그인 (JWT 토큰 발급)
• POST /schedules - 일정 생성 (JWT 토큰 필요)
• POST /api/videos - 비디오 생성 (JWT 토큰 필요)

💬 채팅 (JWT 인증 필요)

📍 region_2depth_name 기반 동적 채팅방 생성
• GET /api/chat?latitude={lat}&longitude={lng} - 사용자 위치로 적절한 roomId 반환
• GET /api/chat/{roomId}/messages - 채팅방 메시지 조회
• POST /api/chat/{roomId}/messages/send - 메시지 전송
• 카카오 주소 API로 region_2depth_name 추출 후 동적 채팅방 생성
• 채팅방 이름: "{region_2depth_name} 채팅방" (예: "안양시 채팅방", "용인시 채팅방")
• 기존 지역 채팅방이 없으면 자동으로 새로 생성
