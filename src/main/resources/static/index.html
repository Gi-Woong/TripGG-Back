🔗 API 엔드포인트 테스트

👥 사용자 관리 (기존)

GET /users (전체 사용자) GET /users/{id} (사용자 상세) GET /users/kakao/{kakaoId}
📝 참고: 새로운 API 구조
• 기존 /users 엔드포인트는 유지
• 새로운 /api/user/* 엔드포인트가 추가됨
• 인증이 필요한 API는 세션토큰 필요

📅 일정 관리

GET /schedules (전체 일정) GET /schedules/{id} (일정 상세)

🏛️ 장소 관리

GET /places (전체 장소) GET /places/{id} (장소 상세)
🏷️ 카테고리 기반 장소 검색 API
• GET /api/places/search - 카테고리별 장소 검색 (카카오맵 API 사용)
• GET /api/places/{placeId} - 장소 상세 조회
• 카테고리 코드: MT1(대형마트), CS2(편의점), FD6(음식점), CE7(카페), HP8(병원), PM9(약국) 등
🔍 카테고리 검색 (가장 확실한 방법)
• 카테고리만으로도 정확한 결과 반환
GET /api/places/search?category=FD6 (음식점) GET /api/places/search?category=CE7 (카페) GET /api/places/search?category=MT1 (대형마트) GET /api/places/search?category=CS2 (편의점)
📍 위치 기반 카테고리 검색
• 특정 지역에서 카테고리별 장소 검색
GET /api/places/search (강남구 음식점) GET /api/places/search (강남구 카페) GET /api/places/search (서울역 음식점)

🎥 비디오 관리

GET /api/videos (전체 비디오) GET /api/videos/{id} (비디오 상세)

🔐 인증 & 회원 관리

GET /auth/kakao/login (카카오 로그인)
📋 POST API 엔드포인트
• POST /api/auth/login - 카카오 토큰으로 로그인
• POST /api/auth/signup - 이메일, 닉키임으로 회원가입
• PUT /api/user/profile/update - 프로필 정보 수정
GET /api/user/profile (내 정보 조회) GET /api/user/video-list (내 영상 목록) GET /api/user/video/detail/{videoId} (영상 상세)

💬 채팅

GET /api/chat/rooms (모든 채팅방)
📍 region_2depth_name 기반 동적 채팅방 생성
• GET /api/chat?latitude={lat}&longitude={lng} - 사용자 위치로 적절한 roomId 반환
• GET /api/chat/{roomId}/messages - 채팅방 메시지 조회
• POST /api/chat/{roomId}/messages/send - 메시지 전송
• 카카오 주소 API로 region_2depth_name 추출 후 동적 채팅방 생성
• 채팅방 이름: "{region_2depth_name} 채팅방" (예: "안양시 채팅방", "용인시 채팅방")
• 기존 지역 채팅방이 없으면 자동으로 새로 생성
GET /api/chat (수원시 테스트) GET /api/chat (새로운 지역 테스트)
